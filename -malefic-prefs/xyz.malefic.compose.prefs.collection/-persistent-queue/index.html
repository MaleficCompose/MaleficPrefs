class PersistentQueue<T : Serializable>(key: String, prefs: Preferences = Common.prefs) : LinkedList<T> , Queue<T>

A persistent implementation of a Queue that saves its state to preferences.

Parameters

the type of elements in this queue, which must be Serializable

the key used to store the queue in preferences
